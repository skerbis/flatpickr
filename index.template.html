üìù All Available Features

  • Dependency-free (no jQuery, Bootstrap, or moment.js required)
  • Simple, polished UX
  • Date + time input with customizable formats
  • Range selections with rangePlugin
  • Multiple date selections
  • Time picker only mode (noCalendar)
  • Human-friendly date formats (altInput)
  • Disable specific dates, date ranges, or dates using custom logic
  • Week numbers display
  • 51 locales available
  • 10 beautiful themes (dark, light, material, glassmorphism, etc.)
  • Numerous plugins (confirm, monthSelect, weekSelect, scroll, etc.)
  • Mobile-friendly with native input fallback
  • Inline calendar mode
  • Min/Max date restrictions
  • Keyboard navigation support
  • Custom date parsing and formatting
  • Event hooks (onChange, onOpen, onClose, etc.)

üîå Plugin Usage Examples

Available Plugins:
  • rangePlugin - Select date ranges
  • confirmDatePlugin - Add confirm button to calendar
  • monthSelectPlugin - Month and year picker
  • weekSelectPlugin - Week selection
  • scrollPlugin - Scroll to increment time
  • minMaxTimePlugin - Time restrictions
  • labelPlugin - Better label support
// Using multiple plugins together
flatpickr("#my-input", {
  enableTime: true,
  plugins: [
    new confirmDatePlugin({
      confirmText: "Confirm",
      showAlways: true
    }),
    new scrollPlugin({})
  ]
});
